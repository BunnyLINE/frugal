# frugal-java

Java library for [Frugal](https://github.com/Workiva/frugal).

## Benchmarking

Benchmarks using [JMH](http://openjdk.java.net/projects/code-tools/jmh/)
are located in the `benchmarks` directory.

```bash
$ # Run benchmarks
$ cd benchmarks/
$ mvn clean instal
$ java -jar target/benchmarks.jar
```
